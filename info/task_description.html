The cakes are laying on the floor. All cakes are round. But our robots can't ignore this fact and trying to find an appropriateness for cakes' disposition. Some cakes form rows and we want to count these rows. The row is a sequence of three or more cakes if we can to draw straight line through its centers. The greater row takes up the smaller rows. So if we have the row with 4 cakes, then we have only one row (not 4 by 3).

The cakes location is represented as a list of coordinates. A coordinate is a list of two integers. You should count the rows.

( Explain input and output values )

Input: A list of coordinates, lists with two integers.

Output: The quantity of rows.

Example: 

checkio([[3,3],[5,5],[8,8],[2,8],[8,2]]) == 2
checkio([[2,2],[2,5],[2,8],[5,2],[7,2],[8,2],[9,2],[4,5],[4,8],[7,5],[5,8],[9,9]]) == 12
